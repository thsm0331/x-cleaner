# x-cleaner by t_t
Chrome extension to delete X (Twitter) history
MIT license

-------

### 설명문
X(구 트위터) 자동 삭제기입니다.  
시간은 더 줄일수 있지만, X 정책에 위반하여 계정이 벤이 되는 경우가 발생한다고 합니다.  
그러니 가급적 기본 시간을 유지하는 것을 권장합니다.  
압축파일 해제 > chrome 확장프로그램에서 압축파일로 설치 후 진행하시면 됩니다.  
로그인 로그 및 삭제 로그는 제공되지 않습니다.  
피드백 내용은 thsm0331@gmail.com 으로 연락 부탁드리겠습니다.  
개인 사용 목적으로 만든 프로그램으로, 기록용으로 올려둔 내용이라 오류 및 피드백이 늦을 수 있다는 점, 양해 부탁드리겠습니다.  
감사합니다.
